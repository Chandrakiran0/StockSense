# StockSense

**StockSense — Inventory Replenishment & Scenario Intelligence System**

Developed by **Chandra Kiran** as an **MBA Final Year Project (Working Model)**.

## Project Overview
StockSense is a Python + Streamlit based decision support system for inventory replenishment planning.  
It helps analyze:
- Reorder Point (ROP)
- Safety Stock
- Suggested Order Quantity
- Inventory Value at Risk
- Scenario Analysis under:
  - Demand Surge (+20%)
  - Lead Time Disruption (+30%)
  - Service Level Upgrade

## Features
- Upload inventory Excel file
- Base case replenishment analysis
- Scenario intelligence dashboard
- Financial exposure comparison
- Procurement burden comparison
- Download full multi-sheet scenario analysis output
